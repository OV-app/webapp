## 📥 Installatie

1. **Clone de repository:**
   ```sh
   git clone https://github.com/OV-app/webapp
   cd ovapp
   ```

2. **Installeer de dependencies:**
   ```sh
   npm install
   ```

3. **Start de applicatie:**
   ```sh
   npm start
   ```

4. **Bezoek de applicatie op:** `http://localhost:3000`
